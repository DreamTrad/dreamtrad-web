# Instructions pour patcher manuellement

Une fois que vous avez téléchargé le patch, il faudra d’abord l’extraire car c’est un dossier compressé zip.

- Copiez le dossier "game" du patch dans le dossier du jeu, donc dans le dossier "The Crimson Debt". Normalement, cela devrait remplacer plusieurs fichiers du jeu.
