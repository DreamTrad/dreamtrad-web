# Pour aller plus loin

### Profil

Le **genre** a une influence sur certaines scènes. Les hommes peuvent obtenir des scènes romantiques avec Gina et Stella, les femmes avec Sha-Ming et Remnan.

En outre, Raqio préfère les joueurs non-binaires, Stella préfère les hommes et Sha-Ming préfère les femmes, ainsi que les joueurs non-binaires (un peu moins).

La couleur a aussi une influence sur l'affection que chaque personnage vous portera. Chaque personnage est représenté par un symbole astrologique (visible dans les Succès ou la Galerie, voir ci-dessous). Si votre couleur correspond à ce symbole, le personnage vous appréciera davantage. Les seuls personnages à ne pas avoir de préférence sont Kukrushka et Yuriko.

*Un tableau avec des données précises est [disponible ici](https://www.reddit.com/r/Gnosia_/comments/v7hszu/color_preferences/) (en anglais).*

### Changer vos stats/Galerie

Si vous êtes Bug, rendez visite à Yuriko la nuit. Elle vous permettra de modifier vos stats. Si vous avez déjà terminé le jeu, elle vous permettra aussi de revoir les illustrations que vous avez obtenues jusqu'ici.
