# Obtenir les différentes fins

## Fin normale

Obtenez toutes les notes. Vous devriez savoir quoi faire.

:::spoiler[l’indice]
Où deviez-vous retourner, une fois la Clé remplie ?
:::
<br>

:::spoiler[la solution]
Paramétrez 0 Gnosia et 15 membres d'Équipage.
:::

## Vraie fin

Obtenez la fin normale. Tout n'est pas résolu...


:::spoiler[le 1er indice]
Vous voulez revoir Setsu, non ?
:::
<br>

:::spoiler[le 2e indice]
Comment l'atteindre, si vous ne voulez pas effacer ce monde sans Gnosia ?
:::
<br>

:::spoiler[le 3e indice]
Si seulement vous pouviez faire quelque chose dont Setsu est incapable...
:::
<br>

:::spoiler[la solution]
Lancez une nouvelle partie sans écraser votre partie terminée. N'hésitez pas à choisir un pseudo différent !
:::
