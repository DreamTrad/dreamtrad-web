# Instructions pour patcher manuellement PC

Une fois que vous avez téléchargé le patch, il faudra d’abord l’extraire car c’est un dossier compressé zip.

- Copiez le dossier "AI_TheSomniumFiles2_Data" du patch dans le dossier du jeu, donc dans le dossier "AI The Somnium Files 2". Normalement, cela devrait remplacer plusieurs fichiers du jeu.
